# 基于SpringBoot+Vue3的图书管理系统

![Spring Boot](https://img.shields.io/badge/-Spring%20Boot-6DB33F?style=flat-square&logo=spring-boot&logoColor=white) ![Vue3](https://img.shields.io/badge/-Vue3-grey?style=flat-square) ![ElementPlus](https://img.shields.io/badge/-ElementPlus-grey?style=flat-square) ![MySQL](https://img.shields.io/badge/-MySQL-4479A1?style=flat-square&logo=mysql&logoColor=white) 

## 项目介绍

基于SpringBoot+MyBatis+Vue3开发的图书管理系统，系统分为管理员端和读者端，管理员可进行图书信息、类型、借阅记录、公告、用户的全面管理，读者可搜索图书、借阅、收藏、修改个人信息等。

**功能模块**
- **首页仪表盘**：展示图书总数、类型数、借阅记录、注册用户数等数据统计，以及图书卡片浏览
- **图书信息管理**：支持图书的增删改查、封面上传、按书名/作者/类型分页搜索
- **图书类型管理**：管理员维护图书分类体系，支持类型的新增、编辑、删除
- **借阅信息管理**：记录图书借阅与归还状态，支持分页查询和条件筛选
- **公告管理**：发布和管理系统公告，读者和管理员均可查看
- **用户管理**：管理员对注册用户进行管理，支持角色区分（管理员/读者）
- **我的收藏**：读者可收藏感兴趣的图书，方便后续查阅
- **个人信息**：用户查看和修改个人资料及密码

**技术架构**
- **后端**：Spring Boot 2.5.6 + MyBatis 2.0.0 + MySQL 8.0
- **前端**：Vue 3.4 + Element Plus 2.5 + Vite 5 + ECharts 5.5 + Pinia 2.1
- **数据库**：MySQL 8.0
